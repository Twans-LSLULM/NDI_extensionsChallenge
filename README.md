# Extension SenSaaSionnel

## Installation

-   Récupérer le code source sur votre machine
-   Ouvrir le gestionnaire d'extensions de votre navigateur (chrome://extensions/ pour Chrome)
-   Activer le mode développeur
-   Cliquer sur "Charger l'extension non empaquetée"
-   Sélectionner le dossier contenant le code source de l'extension

## Permissions

L'extension ne nécessite pas de permissions particulières pour fonctionner.

## Fonctionnalités

-   **Bloc-notes** : Permet de prendre des notes rapidement. Celles-ci sont sauvegardées automatiquement dans le stockage local du navigateur.
-   **TO-DO List** : Une liste de tâches simple où vous pouvez ajouter, marquer comme terminées et supprimer des tâches. Les tâches sont également sauvegardées dans le stockage local.
-   **Minuteur** : Un minuteur simple qui peut être réglé en incréments de 30 secondes. Affiche le temps restant au format HH:MM:SS.
